# 安全措施说明

## 1. 设备绑定许可证

### 实现方式
```
硬件信息 → SHA256指纹 → 许可证绑定 → 无法复制
```

### 设备指纹组成
- CPU序列号
- 磁盘序列号
- 主板序列号
- 计算机名

### 安全性
- ✅ 绑定硬件，无法复制
- ✅ 支持有效期控制
- ✅ 支持功能授权
- ✅ 支持使用次数限制

---

## 2. 使用频率限制

### 默认配置
| 限制项 | 值 |
|--------|-----|
| 每小时上限 | 20次 |
| 每日上限 | 50次 |
| 最小间隔 | 10秒 |

### 防护效果
- 防止批量滥用
- 防止自动化脚本
- 控制API成本

---

## 3. 审计日志

### 记录内容
```
时间戳 | 用户 | 操作 | 资源 | 结果
```

### 记录操作
- 应用启动/关闭
- 许可证导入
- 文件上传
- 检查执行
- 报告导出
- 配置变更

### 安全特性
- ✅ 所有日志自动脱敏
- ✅ 文件路径隐藏完整路径
- ✅ 敏感信息替换为占位符

---

## 4. API密钥加密存储

### Windows平台
```
明文密钥 → DPAPI加密 → 绑定用户 → 安全存储
```

### 特点
- 使用Windows DPAPI
- 绑定当前Windows用户
- 其他用户/设备无法解密

### 非Windows平台
- 使用Base64编码（较低安全性）
- 建议仅用于开发测试

---

## 5. 日志脱敏

### 脱敏规则
| 类型 | 原始 | 脱敏后 |
|------|------|--------|
| 手机号 | 13812345678 | [手机号] |
| 身份证 | 123456789012345678 | [身份证号] |
| 金额 | 12,345.67元 | [金额] |
| 公司 | XX科技有限公司 | [公司名] |
| 邮箱 | test@example.com | [邮箱] |
| IP | 192.168.1.1 | [IP地址] |
| 密钥 | abc123...xyz | [密钥] |

### 文件路径脱敏
```
原始: C:\Users\张三\Documents\投标文件.docx
脱敏: [文件]/投标文件.docx
```

## 安全建议

### 生产环境
1. ✅ 启用所有安全功能
2. ✅ 使用Windows平台（DPAPI加密）
3. ✅ 定期清理审计日志
4. ✅ 为每个用户生成独立许可证

### 开发测试
1. 可以禁用频率限制
2. 可以使用非加密存储
3. 建议使用测试API密钥
